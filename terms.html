2. Descripción del servicio

Nexus OS es una aplicación web de productividad personal que permite a los usuarios gestionar tareas, finanzas personales y notas a través de un sistema unificado de "nodos". El servicio incluye:

  • Panel de control con parser semántico para clasificación automática de contenido.
  • Vista Kanban con gestión de tareas estilo Trello.
  • Módulo de finanzas personales con registro de ingresos y gastos.
  • Sistema de notas con búsqueda por supertags.
  • Vista de calendario para organización temporal.
  • Almacenamiento seguro en la nube con Row Level Security.
